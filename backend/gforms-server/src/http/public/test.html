API Tokens (MCP Inspector)

Generate bearer tokens to use with MCP Inspector or other API clients. Tokens are valid for 24 hours.

Click "Generate New Token" to create a bearer token for MCP Inspector

Using with MCP Inspector

  1. Generate a token using the button above
  2. Copy the token value
  3. Run MCP Inspector: npx @modelcontextprotocol/inspector
  4. Configure connection:
    • URL: https://gforms-mcp.daffyos.in
    • Add header: Authorization: Bearer <your-token>
  5. Connect and test all MCP tools!
